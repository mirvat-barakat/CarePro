<img src="./readme/Titles/title6.svg"/>

> To set up CarePro locally, follow these steps:

### Installation

Clone the repo

```sh
git clone https://https://github.com/mirvat-barakat/CarePro
```

### CarePro Backend

In `backend`:
In The Terminal Run:

```sh
  npm install
```

```sh
  nodemon .
```

### CarePro frontend

In `frontend` :

In The Terminal Run:

```sh
  npm install
```

```sh
  npm start
```
